## CMS功能详解

### 权限管理
* 功能描述

    ```
    更好的管理使用CMS
    分为两种用户类型：
    	1.超级管理员：可以看到所有的菜单
    	2.普通用户：根据权限的设置看到相应的菜单
    登录方式：
    	公司域账号和密码
    ```
* 功能设定

    ```
    主要是两种方式的权限管理：
    	1.管理员可以通过界面操作，在菜单 '设置－账号管理' 中进行添加和编辑
    	2.可以通过配置文件进行添加和编辑 $config['administrator']
    	（该配置有外层覆盖）
    	
    超级管理员：
    	权限仅可通过配置设定
    		$config['administrator']['cms']和$config['administrator']['admin']的合集，拥有超级管理员的权限
    		
    普通用户：
    	以下权限可以由管理员通过界面进行设定：
    		1.功能（群发功能、高级功能）
    		2.管理（消息管理、素材管理、举报管理、自定义菜单管理）
    		3.设置（账号信息、账号管理）
    	
    	以下功能可通过配置文件设定：
    		1.welcome广告（添加、管理）
    		2.banner（添加、查看）
    		3.web页（添加、查看）
    ```
* 功能实现

    

    ```
    1.获取用户权限（包括后台管理和配置管理的两类权限）
    2.取出url中的模块名称 eg：http://chatcms.anjuke.com/{setting}/accountinfo?service_id=100
    	setting为模块名称
    3.判断模块是否在用户管理范围内
    ```
