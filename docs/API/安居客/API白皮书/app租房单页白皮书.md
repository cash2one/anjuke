## app租房单页白皮书

### 1、页面URL
```
打开安居客app ---> 点击首页的`租房`按钮进入租房列表页 ---> 点击一套房源进入租房单页
```

### 2、页面说明
租房单页主要有以下接口：

* 2.1、单页信息接口（包括扣费逻辑）
* 2.2、单页推荐接口－猜你喜欢
* 2.3、单页推荐接口－看了又看
* 2.4、获取tw链接（用于分享？）
