# APP9.0接口设计

### 统一出参结果
| 参数名称	| 类型	| 说明	|
| --------- | ----- | ----- |
| status | int	| 接口返回状态码，0为成功，非0为失败 |
| msg	 | string	| 接口返回信息，根据status不同，0为成功，非0返回错误原因|
| data  	| mixed	| 返回结果|

#### 错误返回格式：

```
{
    "status": "10000",
    "msg": "参数错误",
    "data": {}
}
```
