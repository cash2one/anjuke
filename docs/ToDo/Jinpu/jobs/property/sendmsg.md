
根据用户行为推送信息的job信息  
=====================
------------------
##1 . 根据楼盘价格推送信息

####1 . job作用

```
* 楼盘出租/出售价格从无到有时推送信息(意向用户：pmt14380 意向客户是指用户三个月内订阅，报名团购，报名活动，拨打电话等在该楼盘留下的手机号码)
* 当新盘新铺在crm后台进行价格更改的时候，对意向用户进行信息推送。
* 建议暂停此job

```
####2 . 执行时间

```
每两个小时(现在停止，等待产品介入)
```

####3 . 执行命令
```
php launcher.php Property_SendMsg --action=send_deprice
```
#### 4 . 数据
* 楼盘价格表：`jinpu_db.l_property_price`

* 已发送信息日志表：`l_push_msg`

* 动态表：`jinpu_db.e_news_title`
 
* 活动报名表：`jinpu_db.e_news_registers`（开发商楼盘已经不做，所以此处可忽略废弃）
 
* 400拨打电话记录表：`l_call_info_log`

####5 . 参数定义


参数|描述|
:---------------|:---------------|
action | 发送信息类型

####6 . 数据格式
* 发送信息例子：

```
您关注的祥昭大厦近期公开租金45.0元/平米*月, 详情请咨询4008902008转79423或登录安居客。

```
